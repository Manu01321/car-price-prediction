# car-price-prediction
🚗 Car Price Prediction System is a machine learning web application built with Python, Streamlit, and Scikit-learn. It predicts car prices based on features like year, engine size, and mileage. The app includes a modern UI and a login/register system for secure access.
⚙️ How to Run the Project

1️⃣ Clone the repository

git clone https://github.com/yourusername/car-price-prediction.git

2️⃣ Navigate to the project folder

cd car-price-prediction

3️⃣ Install dependencies

pip install -r requirements.txt

4️⃣ Run the Streamlit app

streamlit run app.py
